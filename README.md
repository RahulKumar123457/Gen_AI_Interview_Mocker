## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
To run the Project 
<br>
Step 1: npm run build
<br>
Step 2: npm run dev
<br>
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
